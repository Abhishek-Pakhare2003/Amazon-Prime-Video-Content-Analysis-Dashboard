# 🎬 Amazon Prime Video Content Analysis Dashboard

An interactive **Power BI Dashboard** that analyzes Amazon Prime Video’s extensive library of movies and TV shows. This project transforms raw streaming data into meaningful visual insights, helping users explore content distribution, audience ratings, global availability, and historical release trends.

# 🚀 Project Overview

This dashboard provides a detailed analysis of Amazon Prime Video’s catalog containing **9,655 titles** released between **1920 and 2021**.

The project focuses on identifying:
- Content distribution between Movies and TV Shows
- Audience rating patterns
- Popular genres
- Historical content growth
- Country-wise availability of titles

The dashboard is designed with a modern dark theme for better readability and executive-level presentation.

---

# 📌 Key Insights

## ✅ High-Level KPIs
The dashboard displays important metrics including:
- **Total Titles:** 9,655
- **Total Ratings:** 25
- **Total Genres:** 519
- **Unique Directors:** 5,771

---

## 🎥 Movies vs TV Shows
A donut chart reveals that Amazon Prime Video is heavily dominated by:

- **Movies:** 80.82%
- **TV Shows:** 19.18%

---

## ⭐ Rating Analysis
The rating distribution chart shows that:
- **13+** and **16+** rated content has the highest share
- Family and teen-focused content dominates the platform

---

## 🎭 Top Genres
The most common genres include:
- Drama
- Comedy
- Documentary
- Action
- Kids Content

---

## 📈 Historical Content Growth
A timeline visualization highlights:
- Slow growth before 2000
- Massive expansion in content after 2010
- Rapid increase in global streaming adoption

---

## 🌍 Country-wise Content Distribution
An interactive world map visualizes:
- Number of shows available by country
- Amazon Prime Video’s global content footprint

---

## 🎛️ Interactive Filters
Users can dynamically filter dashboard data by:
- **Content Type** (Movie / TV Show)
- **Release Year**

This improves dashboard interactivity and user exploration.
